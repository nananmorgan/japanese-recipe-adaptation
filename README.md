# Japanese Recipe Adaptation with Generative AI

## Overview
A web application that adapts Western recipes to use Japanese ingredients and cooking techniques. The system combines GPT-based recipe generation, a fine-tuned ingredient classifier, and a custom "recipe drift" scoring system that measures how far an adapted recipe deviates from the original. Built and deployed as a Streamlit app.

## Live Demo
Try the app here: [Japanese Recipe Drift Checker](https://nanas-recipe-drift-checker.streamlit.app)

## Key Features
- **Recipe Adaptation**: Submit any Western recipe and receive a Japanese-adapted version generated by GPT, with substituted ingredients and modified techniques
- **Ingredient Classification**: A fine-tuned model classifies ingredients as Japanese or non-Japanese to validate adaptations
- **Recipe Drift Score**: A custom metric that quantifies how much an adapted recipe has changed from the original, helping users understand the degree of transformation
- **Interactive Web App**: Deployed via Streamlit for easy access and real-time interaction

## How It Works
1. User inputs a Japanese recipe URL (ingredients + instructions)
2. The app sends the recipe to the OpenAI GPT API with a carefully engineered prompt to generate an adapted version
3. The fine-tuned ingredient classifier evaluates the adapted recipe's ingredient list
4. A drift score is calculated by comparing the original and adapted recipes based on how many 'core', 'supporting', or 'optional' ingredients were changed
5. Results are displayed with the adapted recipe, classified ingredients, and drift score

## Methods & Tools
- **OpenAI GPT API** for recipe generation and adaptation
- **Fine-tuned classifier** for ingredient categorization ('core', 'supporting', 'optional')
- **Custom drift scoring algorithm** measuring recipe deviation across ingredients and techniques
- **Streamlit** for web application deployment
- **Python** (pandas, requests) for data processing and pipeline orchestration
- **GitHub** for version control and deployment integration
